# Hyperplane

Hyperplane is an independent fork of [Plane](https://github.com/makeplane/plane), an open-source project management tool.

**This project is not affiliated with, endorsed by, or supported by Plane Software, Inc.**

## What's different

Changes include:
- Custom container image builds via GitHub Actions (GHCR)
- (Future) Enterprise-grade features for internal workflows

## Upstream

This fork tracks the `preview` branch of [makeplane/plane](https://github.com/makeplane/plane). To sync:
```
gh repo sync MaikBuse/hyperplane --branch preview
```
